# 📊 E-commerce Sales Performance Dashboard — Power BI

> Analyzed e-commerce sales data to track revenue, order status, 
> payment methods, and regional performance across Pakistan.

## 🎯 Key Insights
- Total Sales: **449K** across all regions
- **Karachi** leads in order volume among 3 major cities
- **Cash on Delivery** is the most used payment method
- **Pending orders** are highest in Order Status — area for improvement
- **Football & Laptop** are top 2 selling product categories
